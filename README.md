aura
====

An application for displaying an animated feed from social media

Application Dependencies:
 - libcurl
 - SDL
 - GLEW
 - openssl

Filtering Tool Dependencies:
 - wxWidgets

Plugin depencies:
 - Twitter:
   - liboauthcpp
   - libjsoncpp
 - Facebook:
 - Instagram:
 - Streaming support:
   - libdlna

Packages to install for Ubuntu:
 - g++
 - CMake
 - libsdl2-dev
 - libcurl4-openssl-dev
